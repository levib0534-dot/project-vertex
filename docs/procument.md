# Procurement Guide

To keep the Vertex build accessible, we recommend sourcing parts from industrial suppliers rather than boutique keyboard shops.

## Parts List
- Controller: 2x Seeed Studio XIAO nRF52840.
- PCB: 1.6mm FR4 (ordered from JLCPCB using the provided gerbers).
- Switches: 42x Kailh Choc V2 or V3 (MX-Stem compatible).
- Keycaps: 42x Thick PBT (DSA or XDA profile recommended).
- Battery: 2x 3.7V LiPo 302030 (approx. 150mAh).
- Hotswap Sockets: 42x Kailh Choc Sockets.
- Diodes: 42x SOD-123 (1N4148W).
- Hardware: M2 Standoffs (8mm) and screws.

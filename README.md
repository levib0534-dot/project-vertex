# PROJECT: VERTEX 
> Ergonomic Health. Traditional Soul.

Vertex is an open-source, 42-key wireless split keyboard designed to bridge the gap between high-end custom mechanical acoustics and ergonomic health. 

## Key Features
- Acoustic-First Design: Engineered FR4 sandwich construction for a premium sound signature.
- Low Profile + MX Stems: Optimized for Kailh Choc V2/V3, allowing the use of standard MX keycaps.
- Truly Wireless: Powered by ZMK and Seeed Studio XIAO nRF52840.
